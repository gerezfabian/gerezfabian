Resumen visual del libro "Pro Git"

Esta infografía visualiza el contenido del libro "Pro Git", estructurándolo como un viaje de aprendizaje desde los conceptos fundamentales hasta la maestría interna del sistema.

FUNDAMENTOS: La Filosofía de las Instantáneas (Capítulos 1-2)

Visual: A la izquierda, un bloque de inicio. Muestra una comparación visual clave: "Otros VCS (Deltas)" como una cadena de pequeños cambios, versus "Git (Instantáneas)" como una serie de imágenes completas del sistema de archivos.

El Ciclo de los Tres Estados: Un diagrama circular prominente que muestra el flujo de archivos: Directorio de Trabajo (Modificado) → git add → Área de preparación (Staged) → git commit → Directorio Git (Confirmado).

EL NÚCLEO: Dominando las Ramas (Capítulo 3)

Visual: Un árbol central grande con ramas divergentes y convergentes.

Conceptos Clave: Etiquetas que explican "Ramas = Punteros Ligeros (No copias)". Muestra visualmente una "Fusión Fast-forward" y una "Fusión de 3 vías".

COLABORACIÓN: El Mundo Distribuido y GitHub (Capítulos 4-6)

Visual: Una red de iconos de computadoras interconectadas. Un nodo central grande representa "GitHub/Servidor".

Flujos de Trabajo: Diagramas que muestran "Flujo Centralizado" y "Modelo de Integración/Forking".

Ecosistema GitHub: Iconos para "Pull Requests", "Issues" y "Fork".

HERRAMIENTAS AVANZADAS: La Caja de Herramientas del Experto (Capítulo 7)

Visual: Una caja de herramientas abierta con herramientas especializadas.

Funciones: Stashing, Reescribir la Historia (Rebase Interactivo, Commit --amend), Reset Desmitificado (Soft, Mixed, Hard).

BAJO EL CAPÓ: Fontanería e Internos (Capítulos 8-10)

Visual: Una vista de rayos X o corte transversal de una carpeta .git.

Estructura: Diferencia entre "Porcelana" y "Fontanería". Diagrama de la base de datos de objetos: Commit → Tree → Blobs.
